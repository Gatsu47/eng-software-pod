4  * \par
5  * The PCR manages the accesses to the peripheral registers and peripheral
6  * memories. It provides a global reset for all the peripherals. It also supports the
7  * capability to selectively enable or disable the clock for each peripheral
8  * individually. The PCR also manages the accesses to the system module
9  * registers required to configure the devices clocks, interrupts, and so on. The
10  * system module registers also include status flags for indicating exception
11  * conditions, resets, aborts, errors, interrupts.
