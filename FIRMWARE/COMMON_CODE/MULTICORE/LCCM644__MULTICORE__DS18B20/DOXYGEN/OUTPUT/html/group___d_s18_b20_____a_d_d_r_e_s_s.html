Lint16 s16DS18B20_ADDX__Search ( void  )

Search for all devices on the bus on all channels.

Note
This is a single blocking function that will try and search for everything via binary tree searching of the ROM ID's. THIS IS VERY SLOW. Several hundred sensors can take minutes to search. If this is a problem, use the state machine based search.
